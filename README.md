# tmux-fzf-url

Fork of [wfxr/tmux-fzf-url](https://github.com/wfxr/tmux-fzf-url).

### [TPM](https://github.com/tmux-plugins/tpm)

Add this line to your tmux config file, then hit `prefix + I`:

``` tmux
set -g @plugin 'junegunn/tmux-fzf-url'
```

### Usage

Press `PREFIX` + `u`.
